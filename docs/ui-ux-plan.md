> **更新**：2026-06-30

## 1. 目標

在不大改視覺語言的前提下，讓帳本在 1440px 筆電、平板與規則頁閱讀體驗更穩定，並降低主頁維護成本。

## 3. 1440px 表格策略

- 表格仍允許橫向捲動（欄位多、不宜硬砍欄）
- 在 1440px 以下：縮小 sticky 前三欄寬度、整體 min-width 1180px、cell padding 略減
- 768px 以下維持既有 `@media (max-width: 768px)` 橫向捲動行為

---

## 4. 規則頁文案原則

- 產品名稱：**Ivy的獎金**（非泛稱 AM 系統）
- 一句話：**回簽決定怎麼算，收款決定何時發**
- 完整制度以官方 V2 文件為準，本頁只做帳本試算摘要
