## 3. RWD 策略

### 1440px 表格

- 表格仍允許橫向捲動（欄位多、不宜硬砍欄）
- 在 1440px 以下：縮小 sticky 前三欄寬度、整體 min-width 1180px、cell padding 略減

### ≤768px 手機

- **側欄**：預設隱藏；左上角固定 ☰ 按鈕 → 左側 drawer 滑出完整側欄（品牌、三頁導覽、改密、登出）
- **QuarterContextBar**：垂直堆疊（標題 → 篩選 → KPI）
- **ledger-tabs**：2×2 grid，不用橫向捲動列
- **動畫**：停用 blur 入場、panel shimmer，減少 GPU 閃爍

- 完整制度以官方 V2 文件為準，本頁只做帳本試算摘要
